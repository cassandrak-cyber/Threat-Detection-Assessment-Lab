# Threat Detection Assessment Lab

## Objective

Simulate real-world attacker behavior in a controlled environment and document detection opportunities.

## Attack Scenarios

1. Network Reconnaissance
2. Port Scanning
3. Web Enumeration
4. Credential Attacks
5. SMB Enumeration

## Detection Opportunities

1. Network Traffic Analysis
2. Event Log Analysis
3. MITRE ATT&CK Mapping
4. IOC Development
5. Defensive Recommendations
